Eclipse use & code convention

Eclipse use

项目导出压缩包

File->Export->General->选择要压缩的类型

修改缩进等格式(CTRL+SHIFT+F)

选中代码->Source->Format

添加javadoc注释（ALT+SHIFT+J)

选中元素->Source->Generate Element Comment

改名(ALT+SHIFT+R)

选中需要改名的元素->右键Refactor->Rename
.将会修改文件中所有有这个名字的元素

生成javadoc

code convention

java code convention

  • Javadoc comment
    • Header/Classes
    • Functions
  • Name
    • Package: student
    • Class&Inteface: Student
    • variable&method: inputFileSize
    • constant:MAXWEIGHT
  • Layout/indentation
  • Space
